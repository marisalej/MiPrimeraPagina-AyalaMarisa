Acerca de mí

Hola, mi nombre es Marisa Ayala. Este proyecto fue desarrollado como parte del Curso Python Flex dictado en la plataforma de Coderhouse durante los meses Oct/25-Ene/26.

IberaGym es una aplicación web pensada para la gestión de socios, entrenadores y planes de un gimnasio, aplicando conceptos como CRUD, autenticación de usuarios, vistas basadas en clases y diseño con Bootstrap.

Tecnologías utilizadas:

  • Python
  • Django
  • HTML & Bootstrap 5
  • SQLite
